
Install: copy files to ~/Documents/Bitwig Studio/Library/Presets

Description:
 * BW Native OTT is Ableton's Multiband Dynamics OTT preset recreated in Bitwig's Multiband FX-3 + Dynamics. For some reason, putting the exact same values that Ableton has into Bitwig sounds different, so I added macros to offset the attack and release to get the same sound out while keeping the same Dynamics settings.
